Colors

Accessible colors are exposed in the scheme via the appropriate with-accessible-* modifier. Background/Foreground ratios meet the WCAG AAA Color contrast requirements. However, keep in mind that if you use a color on a different background you'll need to make sure it's WCAG AAA compliant.

The following schema is used for color type classes:

  • ee-{color}-color: For text color.
  • ee-{color}-background-color: For background color
  • with-accessible-background: used with the ee-*-color class to add an accessible background color.
  • with-accessible-foreground: Used with the ee-*-background-color class to add an accessible foreground color.
  • with-linear-gradient: Applies a linear-gradient background for the given color class
  • with-hover: Used on selectors with an applied background to provide an accessible hover state for the background.
  • with-01-transparency: Currently only available for background colors. Applies a transparency value of .01 to the background.
  • with-25-transparency: Currently only available for background colors. Applies a transparency value of .25 to the background.
  • with-50-transparency: Currently only available for background colors. Applies a transparency value of .50 to the background.
  • with-75-transparency: Currently only available for background colors. Applies a transparency value of .75 to the background.

Entity Status Colors

These are status colors for various Event Espresso model entities. The following schema is used for the class names:

  • .ee-status-color-{status_code}: The text will be colored for the given status code.
  • .ee-status-background-color-{status_code}: The background color will be colored for the given status code.
  • .ee-status-color-{status_code}.with-accessible-background: This will add a background color that meets the accessible color contrast ratio.
  • .ee-status-color-{status_code}.with-accessible-background.with-hover: This adds a hover color to the element.
  • .ee-status-background-color-{status_code}.with-accessible-foreground: This will apply a color to the content with an accessible color contrast ratio.
  • .ee-status-background-color-{status_code}.with-accessible-foreground.with-hover: Applies a color for the hover event on the selector.
