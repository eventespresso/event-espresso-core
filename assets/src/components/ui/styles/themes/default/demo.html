Colors

The following schema is used for color type classes:

  • ee-{color}-color: For text color.
  • ee-{color}-background-color: For background color
  • with-dark-contrast: Used to swap in the dark accessible contrast color matching the main color
  • with-bright-contrast: Used to swap in the bright accessible contrast color matching the main color
  • with-linear-gradient: Applies a linear-gradient background for the given color class

Entity Status Colors

These are status colors for various Event Espresso model entities. The following schema is used for the class names:

  • .ee-status-color-{status_code}: The text will be colored for the given status code.
  • .ee-status-background-color-{status_code}: The background color will be colored for the given status code.
  • .with-bright-contrast: Used to swap in the bright contrast accessible color for that status.
  • .with-dark-contrast: Used to swap in the dark contrast accessible color for that status.

strong>Note: For these examples, we're just showing the colors not the text color. For implementation you can use similar technique as demonstrated in the previous color demo.
