Entity Status Colors

These are status colors for various Event Espresso model entities. The following schema is used for the class names:

  • .ee-status-color-{status_code}: The text will be colored for the given status code.
  • .ee-status-background-color-{status_code}: The background color will be colored for the given status code.
  • .ee-status-color-{status_code}.with-accessible-background: This will add a background color that meets the accessible color contrast ratio.
  • .ee-status-color-{status_code}.with-accessible-background.with-hover: This adds a hover color to the element.
  • .ee-status-background-color-{status_code}.with-accessible-foreground: This will apply a color to the content with an accessible color contrast ratio.
  • .ee-status-background-color-{status_code}.with-accessible-foreground.with-hover: Applies a color for the hover event on the selector.
