{{colorLabel}} Theme Color

{{colorLabel}} background color
+ WCAG compliant text color

{{{desc}}}

{{#if isDefault}}
  • --ee-default-text-color - The standard text color used for most elements. The text you are reading right now is utilizing this color variable.
  • --ee-default-text-color-high-contrast - A high contrast text color for use with smaller font sizes.
  • --ee-default-text-color-super-high-contrast - A super high contrast text color for use with really small font sizes.
  • --ee-default-text-color-low-contrast - A low contrast text color for use with larger font sizes.
  • --ee-default-text-color-super-low-contrast - A super low contrast text color for use with really large font sizes.

The default theme style also uses the --ee-background-color CSS variable for setting the default background color for containers, including the background for this entire page.

{{else}}
  • --ee-color-{{color}} - primarily intended for setting the background color of an element
  • --ee-text-on-{{color}} - A WCAG compliant text color for the above.
  • --ee-color-{{color}}-high-contrast - A higher contrast shade of the --ee-color-{{color}} color.
  • --ee-text-on-{{color}}-high-contrast - A WCAG compliant text color for the above.
  • --ee-color-{{color}}-low-contrast - A lower contrast shade of the --ee-color-{{color}} color.
  • --ee-text-on-{{color}}-low-contrast - A WCAG compliant text color for the above.
{{/if}}
